Explanation: Solve by Factoring

To solve a quadratic equation like \(ax^2 + bx + c = 0\) by factoring, you need to find two numbers that multiply to \(c\) (or \(ac\) if \(a \neq 1\)) and add up to \(b\).

For example, to solve \(x^2 - 5x + 6 = 0\):

  1. Find two numbers that multiply to 6 and add to -5. These are -2 and -3.
  2. Rewrite the equation as \((x - 2)(x - 3) = 0\).
  3. Set each factor to zero and solve:
    • \(x - 2 = 0 \Rightarrow x = 2\)
    • \(x - 3 = 0 \Rightarrow x = 3\)

So, the solutions are \(x = 2\) or \(x = 3\).

Lesson: Understanding Quadratic Factoring

Welcome to the lesson on solving quadratic equations by factoring. This method is a fundamental skill in algebra.

A quadratic equation is an equation of the form \(ax^2 + bx + c = 0\), where \(a \neq 0\).

Factoring involves rewriting the quadratic expression as a product of two linear factors. For example, \(x^2 - 5x + 6\) can be factored into \((x-2)(x-3)\).

Once factored, if \((x-p)(x-q) = 0\), then either \(x-p=0\) or \(x-q=0\). This gives us the solutions \(x=p\) or \(x=q\).
